# ES6 Data Manipulation
This project focuses on mastering modern JavaScript (ES6+) data structures and manipulation techniques. It covers everything from basic array operations to advanced memory management with Typed Arrays.

## 🛠️ Technologies & Concepts
Array Manipulation: Using map, filter, and reduce for functional programming.

Typed Arrays: Handling raw binary data with ArrayBuffer and DataView.

Sets: Managing collections of unique values and performing membership checks.

Maps: Using keyed collections for structured data and performing updates.

ES6 Syntax: Arrow functions, modules (import/export), and short-circuit evaluation.

## 🚀 Key Task Summaries
### 🔹 Basic Array Operations (Tasks 0-4)

Learned to process collections of objects using standard ES6 methods:

Map: Transforming objects into simple arrays (e.g., getting IDs).

Filter: Selecting specific data points based on criteria (e.g., location).

Reduce: Aggregating data into a single value (e.g., summing IDs).

### 🔹 Memory Management (Task 5)

Implemented ArrayBuffer and DataView to interact with raw binary data. This task emphasized the importance of bounds checking to prevent memory errors.

### 🔹 Working with Sets (Tasks 6-8)

Explored the Set object for high-performance uniqueness checks.

Developed logic to filter and "clean" set data based on string prefixes.

Implemented subset validation using every() and has().

### 🔹 Advanced Key-Value Pairs (Tasks 9-10)

Utilized Map for structured data storage:

Initialized maps with bulk entries.

Mutated map values by reference using .forEach() and .set().

Implemented error handling to ensure functions only process valid Map instances.
