# JavaScript Object-Oriented Mastery
This repository is a deep dive into advanced Object-Oriented Programming (OOP) in JavaScript. We move beyond simple variables, focusing on how to build robust, scalable class structures using modern ES6 features, symbols, and inheritance patterns.

# Project Scope
- These tasks explore the mechanics of the JavaScript engine, transforming how we structure objects and data.
- Encapsulation & Validation: Protecting data using the _ underscore convention and "Gatekeeper" setters.
- Inheritance & Abstraction: Building class blueprints that enforce a "contract" for subclasses.
- Composition: Combining independent classes to build more complex features.
- Advanced Symbols: Using Symbol.species and Symbol.toStringTag to manipulate how JS engines handle objects.
- Hoisting & The Temporal Dead Zone: Understanding why class definitions must come before instantiation.
- Type Coercion: Teaching objects how to "pretend" to be primitives (Numbers/Strings) when necessary.
