# NoSQL - MongoDB Exercises
This project focuses on mastering NoSQL concepts using MongoDB. The tasks cover the full lifecycle of database interaction, starting from basic administrative tasks in the MongoDB shell to building programmatic solutions using the pymongo Python driver.

# Technologies Used
MongoDB: Version 3.6.3

Python: 3.x

Library: pymongo

Operating System: Ubuntu 20.04 (or similar)

| # | Filename | Description |
|---|---|---|
| 0 | `0-list_databases` | List all databases |
| 1 | `1-use_or_create_database` | Create or use a database |
| 2 | `2-insert` | Insert a document into a collection |
| 3 | `3-all` | List all documents |
| 4 | `4-match` | List documents with a filter |
| 5 | `5-count` | Count documents in a collection |
| 6 | `6-update` | Update a document |

| 8 | `8-all.py` | List all documents (Python) |

| 11 | `11-schools_by_topic.py` | List schools by topic (Python) |
| 12 | `12-log_stats.py` | Analyze Nginx logs in MongoDB |
